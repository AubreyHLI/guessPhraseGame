# guessPhraseGame
Some English sentences are divided into seperated characters, please guess the sentence and combine the words. 

将拆分的英语词汇组合，并还原为一个完整的句子，游戏即成功。每一轮游戏有4次出错机会，如果全猜错，游戏就失败。
